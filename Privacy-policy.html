1. Information We Collect

When you use commands, we process:

  • Search queries: Anime titles, character names, studio names (temporarily cached)
  • Suggestion content: Submitted via /suggest-report (sent to suggestion channel, not stored by bot)
  • Anonymous statistics: Command names, frequency, error counts (no user identifiers)

3. Data Storage & Security

3.1 Storage Location

All data is stored locally on the bot host machine in JSON files within a /data directory. No cloud databases or external storage services are used. The bot host is located in Asia/Manila timezone.

3.2 Security Measures

  • Data files are stored with restricted file permissions
  • Bot token is stored in environment variables (.env), NEVER hardcoded
  • No direct public access to data files
  • Regular automated backups are not performed (data is ephemeral)

3.3 Data Retention

  • AFK Data: Removed when user returns; persistent backup until bot removal
  • News Configurations: Stored until bot is removed from server
  • Meme History: Auto-cleaned after 60 days
  • Cache: Auto-cleaned after 5 minutes (searches) or 7 days (watchtime)
  • Command Logs: No persistent logs of individual commands

4. Third-Party Services

Ani Bot interacts with the following third-party services. Each has its own privacy policy:

  • Jikan API (MyAnimeList): Fetches anime/character data. Your search queries are sent to Jikan's servers.
  • RSS.app: Provides news feeds. No user data is shared with RSS.app.
  • Reddit (r/animemes): Fetches meme images. No user data is shared with Reddit.
  • Discord: Platform itself. We receive Discord IDs and usernames as part of normal API operation.

We do not control these third-party services and are not responsible for their practices.
